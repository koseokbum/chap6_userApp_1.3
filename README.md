# 사용하는 방법

## 실행 

이것은 원본의 `userApp` 앱을 Meteor Version 1.4.2.1로 수정해 본 것이다. 아주 깔끔하지는 않다.


이 깃저장소를 클론닝 한 이후에 다음과 같이 시행한다.


    $ git clone https://github.com/koseokbum/chap6_userApp_1.3.git



그리고 해당 앱 디렉터리로 이동한 다음, 다음을 실행한다.


    $ meteor npm install
    $ meteor run


## 바꾼 부분
